# 個人作品集網站

一個參考 brittanychiang.com 風格設計的個人作品集網站，使用 HTML 和 Tailwind CSS 建立。

## 功能特色

- ✅ 深色主題設計（海軍藍背景）
- ✅ 響應式佈局（左側導航 + 右側內容）
- ✅ 關於我區塊
- ✅ 作品展示區（Grid 佈局）
- ✅ 聯絡資訊區塊
- ✅ 平滑滾動導航
- ✅ 社交媒體連結

## 設計特點

- **配色方案**：深藍色背景（`#0a192f`）+ 青色強調色（`#64ffda`）
- **佈局**：左右分欄設計，左側固定導航，右側可滾動內容
- **字體**：使用 Inter 字體，提供清晰的閱讀體驗
- **互動效果**：懸停效果、平滑過渡動畫

## 使用方法

### 方法一：直接開啟 HTML 檔案

1. 雙擊 `index.html` 檔案
2. 瀏覽器會自動開啟網站

### 方法二：使用本地伺服器

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

然後在瀏覽器訪問 `http://localhost:8080`

## 自訂內容

### 修改個人資訊

編輯 `index.html` 檔案中的以下部分：
- 標題和副標題（第 70-73 行）
- 關於我內容（第 100-120 行）
- 專案資訊（第 125-220 行）
- 聯絡資訊（第 225-260 行）

### 修改配色

在 `index.html` 的 `<script>` 區塊中（第 8-25 行）修改 Tailwind 配置的顏色。

## 技術棧

- **HTML5**: 結構標記
- **Tailwind CSS**: 樣式框架（透過 CDN）
- **Vanilla JavaScript**: 基本互動功能

## 瀏覽器支援

- Chrome（最新版本）
- Firefox（最新版本）
- Safari（最新版本）
- Edge（最新版本）

## 注意事項

- 此專案使用 Tailwind CSS CDN，需要網路連線才能載入樣式
