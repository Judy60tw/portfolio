範例三：記憶翻牌遊戲 (Memory Match Game)
這個遊戲非常適合練習 JavaScript 的陣列處理 (Array Shuffling) 以及 非同步操作 (setTimeout)。

遊戲玩法：畫面上有多張蓋住的卡牌，玩家每次翻開兩張，若圖案相同則配對成功並保持翻開；若不同則在短暫延遲後翻回去。

技術要點：

使用 Array.sort(() => Math.random() - 0.5) 進行洗牌。

利用 dataset 屬性儲存卡牌類型。

練習處理 CSS 動畫（翻轉效果）與 JS 的邏輯同步。

在 Cursor 中建議輸入的 Prompt：

「請用 HTML, CSS, Vanilla JS 製作一個 4x4 的記憶翻牌遊戲。背景使用米色，卡牌翻開時有平滑動畫。請確保點擊太快時不會產生錯誤，並在配對完成時顯示恭喜訊息。」
